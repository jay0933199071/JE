雙狹縫干涉

理解雙狹縫實驗

雙狹縫實驗是物理學中一個著名的實驗，它展示了光和物質既可以像粒子一樣，也可以像波一樣的特性。在這個實驗中，光通過兩個非常靠近的狹縫，然後在後方的屏幕上形成條紋圖案，這是一種波的干涉現象。

原理

當光穿過兩個狹縫時，它像波一樣擴散開來，然後相互干涉。這種干涉會在屏幕上形成一系列明亮和黑暗的條紋，稱為干涉圖案，這只能用波的性質來解釋。

干涉的數學公式

干涉圖案由下面的公式描述：
nλ = d sin(θ)
其中：

  • n 是亮條紋或暗條紋的序數。
  • λ 是光的波長。
  • d 是兩個狹縫之間的距離。
  • θ 是干涉條紋的角度。

概念/項目 詳細說明
雙狹縫干涉實驗 在一個屏幕上有兩個狹縫，光源發出的光波通過這兩個狹縫後，會在螢幕上形成干涉條紋。這是波動性質的一個證據。
干涉條紋 由兩個狹縫發出的光波相互干涉而產生的條紋。通常是明條紋和暗條紋。
條紋的形成原理 當兩束光波相遇時，如果它們的波峰與波峰、波谷與波谷重合，則會產生建設性干涉，形成明條紋；如果波峰與波谷重合，則會產生破壞性干涉，形成暗條紋。
干涉條紋的間距公式 Δy = λL / d
符號解釋
  • Δy：干涉條紋的間距
  • λ：光的波長
  • L：觀察屏幕距離狹縫的距離
  • d：兩狹縫之間的距離
最大強度條紋位置 ym = m λ L / d (其中 m = 0, 1, 2, …)
最小強度條紋位置 y_min = (m + 1/2) λ L / d (其中 m = 0, 1, 2, …)
條紋序數 m 是條紋的序數，m = 0 為中心明條紋，其他的為亮條紋，m 可以是正整數或零。
光源條件 使用相干光源，保持兩個狹縫之間的光波相位穩定。
波長的影響 波長越長，干涉條紋間距越大。
雙狹縫干涉的限制 若狹縫過窄或光源過遠，則可能無法清晰觀察干涉現象。狹縫的寬度與光源的寬度也會影響干涉效果。
